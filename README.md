# github-activityCLI

Created for practicing Python url requests, json parsing, and CLI development. Display recent GitHub activity for a specified user.

## Usage

1. Clone the repository: `git clone https://github.com/apathydevil/github-activityCLI`
2. Navigate to the project directory: `cd github-activityCLI`
3. Install dependencies: `pip install -r requirements.txt`
4. Run the CLI: `python main.py <github_username>`
5. Replace `<github_username>` with the GitHub username you want to fetch activity for.
6. The CLI will display the recent activity of the specified GitHub user in a readable format.
